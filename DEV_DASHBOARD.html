🧪 Laboratoire de Démonstration (Soutenance)

🛡 Résilience & Persistance
Scénario de Test : 1. Supprimer tous les pods backend réels.
2. Regarder ArgoCD les recréer tout de suite (Self-healing).
3. Refaire une commande SQL via Swagger pour prouver que les données n'ont pas disparu (PVC).
kubectl delete pods -l app=backend -n devops-prod
🔥 Observabilité (Stress Test)
Scénario de Test : 1. Lancer 200 requêtes API ultra-rapides.
2. Basculer sur Grafana pour voir les pics de CPU et de débit.
3. Démontrer l'alerte Prometheus si le seuil est dépassé.
Lancer le Stress Test (200 req)
⚙ Pipeline CI/CD (GHA)
Scénario de Test : 1. Faire un commit vide pour simuler un changement de code.
2. Montrer sur GitHub Actions le Build Docker et le Scan Trivy.
3. Observer la synchronisation automatique dans ArgoCD.
Déclencher le Pipeline GitHub
🐥 Canary : Déclenchement (GitOps)
Scénario de Test : Mettre à jour la version du déploiement via Gita. Cela déclenche le Rollout progressif pour éviter une interruption de service totale.
Lancer Mise à Jour Canary (Push Git)
